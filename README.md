A simple and efficient web application to track daily expenses, manage budgets, and analyze spending habits.

---

## ✨ Overview

The Expense Tracker helps users record their income and expenses, categorize transactions, and visualize financial data for better money management.

> 💡 “Track your money, control your future.”

---

## 🚀 Features

- ➕ Add income and expenses  
- 🗑️ Delete transactions  
- 📊 View balance, income, and expenses  
- 📂 Categorize transactions  
- 📅 Track daily spending  
- 📈 Simple and clean UI
